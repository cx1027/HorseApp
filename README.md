# HorseApp

PWA-first Next.js + Supabase MVP for horse ownership, management, and investor reporting.

## Prerequisites
- Node.js 18+
- npm 9+
- Supabase project for production

## Setup
1. Copy `.env.example` to `.env.local`.
2. Fill in Supabase URL and anon key.
3. Run `npm install`.
4. Run `npm run dev`.
5. Open `http://localhost:3000`.

## Notes
- Phase 1 uses in-app notifications only.
- Payments and AI are not included in Phase 1.
- Use the Figma reference for visual direction only.
